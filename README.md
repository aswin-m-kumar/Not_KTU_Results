# Not\_KTU\_results Mock Website

This project is a simple, front-end mock-up of a university e-Governance portal, named **"Not\_KTU\_results"**. It was created for demonstration purposes to replicate the look and feel of a real university portal based on provided screenshots.

---

## 🚀 Features

* **Homepage:** A clean landing page with sections for Institutions, Students, and the University.
* **Login/Register Page:** A separate view for a user sign-in form.
* **Simple Navigation:** Users can switch between the homepage and the login page without a page reload.
* **Responsive Design:** The layout is built with Tailwind CSS to be functional on both desktop and mobile devices.
* **Custom Styling:** The color scheme and layout are designed to closely match the reference images provided.

---

## 🛠️ Technologies Used

* **HTML5:** For the basic structure and content of the web pages.
* **Tailwind CSS:** A utility-first CSS framework for rapid UI development.
* **Font Awesome:** For scalable vector icons.
* **JavaScript:** For basic interactivity, such as switching between the home and login page views.

---

## 📖 How to Use

1. **Clone or download the repository.**
2. Open the `index.html` file in your web browser.
3. You will see the **homepage** by default.
4. Click on the **"Login / Register"** button in the header to switch to the login form view.
5. Click **"Back to Home"** (the same button) to return to the homepage.

---

## ⚠️ Disclaimer

This is **not** a real or functional university website. It is a static front-end project created for educational and portfolio purposes. The login/register form does **not** connect to any backend or database and does **not** handle any real user data.
