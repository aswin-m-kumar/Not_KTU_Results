### Project Description
A hilariously broken university e-governance portal that perfectly mimics real-world university systems by crashing exactly when students need exam results the most.

### The Problem (that doesn't exist)
University portals are too reliable! Students get their results too easily, without the proper amount of anxiety, server timeouts, and existential dread that builds character.

### The Solution (that nobody asked for)
We've created the ultimate "working" university portal that crashes with a 504 Gateway Timeout precisely when exam results are supposed to be available. Features include an impossible password game, sarcastic FAQs, and the authentic university experience of disappointment!

## Key Features That Make This Gloriously Useless

### 🎮 The Progressive Password Game
- Starts innocent with "8 characters minimum"
- Escalates to mathematical nightmares: "Numbers must add up to 10"
- Includes cultural requirements: "Must include a color name"
- Demands temporal awareness: "Must contain current month"
- Ends with classical education: "Must include Roman numerals"
- Features real-time sound effects for maximum frustration

### 💥 Authentic University Experience
- **504 Gateway Timeout**: Appears exactly when you need results most
- **Delayed Loading**: 10-second delay because anticipation builds character
- **Sarcastic Support**: FAQ answers that mirror real university helpdesk energy

### 🔊 Audio Feedback System
- Success chimes for completed requirements
- Error buzzes for failures
- New rule notification sounds
- Victory fanfare before the inevitable crash

### 📱 Responsive Disappointment
- Works on all devices so no one escapes the experience
- Mobile-optimized frustration
- Cross-platform server timeouts

## Why This Project Is Perfectly Useless

It's a perfect simulation of real university systems, making it completely useless for getting actual results but invaluable for building character and managing expectations.
